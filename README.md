
# 智能健康助手

一个面向健身爱好者和健康管理人群的一站式健康管理平台，提供运动记录、饮食记录、体重追踪、智能建议等功能。

## 项目简介

智能健康助手是一个个人项目，旨在帮助用户科学地管理健康数据，提供个性化的健康建议。项目使用 React 18 + TypeScript + Vite 开发，具有现代化的用户界面和良好的用户体验。

## 功能特性

### 核心功能
- ✅ 用户认证：密码登录、记住登录状态
- ✅ 仪表板：健康数据概览、趋势图表、目标进度
- ✅ 运动记录：添加、编辑、删除运动记录，支持多种运动类型
- ✅ 饮食记录：40+种常见食物，按重量计算热量和营养成分
- ✅ 体重追踪：记录体重、体脂率、肌肉量，BMI自动计算
- ✅ 智能建议：基于用户数据的个性化健康建议
- ✅ 数据报表：统计分析、营养成分占比、成就系统
- ✅ 个人中心：个人信息管理、目标设置

### 技术特性
- 🔧 React 18 + TypeScript 类型安全
- 🎨 Tailwind CSS 现代化样式
- 📊 Chart.js 数据可视化
- 🗄️ Zustand 状态管理
- 💾 本地存储持久化
- 📱 响应式设计

## 使用说明

### 登录
- 演示账号：用户名 `user`，密码 `123456`

### 运动记录
1. 点击「添加运动」按钮
2. 选择运动类型，输入时长、消耗热量等
3. 保存记录

### 饮食记录
1. 点击「添加饮食」按钮
2. 选择餐次和食物
3. 输入食物重量
4. 系统自动计算热量和营养成分

### 体重追踪
1. 点击「记录体重」
2. 输入体重、体脂率、肌肉量（可选）
3. 查看体重变化趋势
