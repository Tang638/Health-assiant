## 四、技术要点

### 4.1 状态管理
- 使用 Zustand 进行状态管理
- 支持本地存储持久化
- 状态包括：用户、活动、饮食记录、体重记录、目标、建议等

### 4.2 路由结构
```
/ - 首页（已登录）
/login - 登录页
/activities - 运动记录
/diet - 饮食记录
/weight - 体重追踪
/suggestions - 智能建议
/reports - 数据报表
/profile - 个人中心
```

### 4.3 常用食物库
包含40+种常见食物，按分类筛选，支持按重量计算热量和营养成分

### 4.4 数据可视化
使用 Chart.js 展示：
- 运动趋势图
- 体重变化图
- 营养成分占比图
